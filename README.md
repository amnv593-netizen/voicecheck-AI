# VoiceCheck AI 🎙️
### AI-Powered Voice Fake News Detector

VoiceCheck AI is a Machine Learning web application that detects fake news directly from voice and audio messages.

## How It Works
1. User uploads a voice message (wav, mp3, m4a)
2. Google Speech Recognition converts voice to text
3. Random Forest ML model predicts FAKE or REAL
4. Result shown with confidence percentage

## Tech Stack
- Python
- Scikit-learn (Random Forest Classifier)
- Google Speech Recognition API
- Streamlit
- Pandas
- TF-IDF Vectorization

## Dataset
- 44,000+ American fake news articles
- 56,000+ Indian news articles (IFND Dataset)
- Total: 100,000+ articles

## Features
- Supports Hindi and English voice input
- Shows confidence percentage with result
- Works on WhatsApp style voice forwards
- Built for Indian users
